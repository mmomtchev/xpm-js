# The `minimumXpmRequired` property

## Purpose

Identify the minimum required **xpm** version.

## Syntax

```json
{
  ...
  "xpack": {
    "minimumXpmRequired": "<semver>"
  }
}
```

The value of the `minimumXpmRequired` property is a string with a version in [SemVer](https://semver.org) format.

## Default value

If missing, no checks are performed on the **xpm** version.

<MinimumRequired name="xpm" fileName="package" />

```json {4}
{
  ...
  "xpack": {
    "minimumXpmRequired": "0.19.4"
  }
}
```
