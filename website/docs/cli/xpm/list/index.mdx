# xpm list

## Purpose

List installed dependencies.

## Synopsis

```sh
xpm list [<options> ...]
```

Aliases:

- `ls`
- `ll`

## Description

List the packages installed in the project located
in the current folder or installed in the user global xPacks store.

## Options

### List global packages (`-g|--global`)

List the packages installed in the user global xPacks store.

{/*
### List system packages (`-sy|--system`)

List the packages installed in the system store.
*/}

### Configuration (`-c|--config`)

List the dependencies installed in an existing configuration build folder
instead of the dependencies installed in top project folder.

## Examples

[`@micro-os-plus/utils-lists`](https://github.com/micro-os-plus/utils-lists-xpack/)
project.
