### Prepare the release


- switch to the `development` branch- `npm run fix`
- if there are changes, **commit** all changes into the `development` branch- `npm run test`
- check the latest commits `npm run git-log`
- update `CHANGELOG.md`, add a line like _**\* v0.20.8 published on npmjs.com**_
- **commit** with the message _**CHANGELOG: publish npm v0.20.8**_
- check the list of files included in the npm package:
  ```sh
  npm run pack
  ```
- possibly update `.npmignore`
