### Tag the npm package as `latest`

When the release is considered stable, promote it as `latest`:

<CodeBlock language="sh"> {
`npm dist-tag add xpm@0.20.8 latest
`} </CodeBlock>

<CodeBlock language="sh"> {
`npm dist-tag ls xpm
`} </CodeBlock>

<CodeBlock language="sh"> {
